Text Editor + Speech Recognition with WPF

This project is a text-editor clone I build using WPF (Windows Presentation Foundation), a UI framework included in Microsoft's .NET framework. I used XAML (Extensible Application Markup Language) and C# to design and provide functionality to interactive UI elements, including the ability to format text in conventional ways (Font Styles, Sizes, Bold/Italicized/Underlined Text). This editor also allows text-to-speech functionality, which allows the user to talk, and have their words recognized and displayed in the editor. Text files can also be saved and opened. You can see the code here on my GitHub.
